# 危机边缘
2018年7月25日
15:40

大概来说，世界因为时间穿越的问题经历过3个版本，每个版本里都有甲乙两个平行世界（或者说是平行宇宙）同时存在。三个版本的世界里的人分别称为the first people，the second people，the third people。如果觉得叫“第几个版本的世界”有点变扭，也可以叫他第几次轮回。

大概发展过程如下：
### 一、原始世界（末日机器产生）：

第一个版本的世界里，应该是这边的P死了，那边的Peter被w乙救活了。但是后来这边的W或者其妻子太想念P，所以就穿越了。穿越之后两个世界都遭到了严重破坏。全世界的人都死了。W在死之前想到了办法修复2个世界，但是需要2个世界的人共同合作。于是制造出了那个机器送到过去，以便让Peter用它来建立2个世界的桥梁。可能后来又怕回到过去的人们不知道这个机器的作用，就又送回去了一本书和一张图，以便指引他们使用这个机器。但是书里又不能写的很直白，所谓天机不可泄露。如果人们都知道了自己的未来，那未来就更加不可预知了。

第一个版本的世界至此结束。时间回到了过去。这个版本的世界里没有末日机器，而是在世界灭亡前制造了末日机器送到了过去，这个世界里的人就是the first people。

### 二、第一次回溯的世界（the first people的目的没有达到，然后观察者在这个世界里产生）：

然后第二个版本的世界开始发展。前期的发展和第一个版本的世界基本一致，只是多了一个末日机器，一本书，一张图，但是没有观察者。Peter照样是由另外一个世界的W救活，后来被带到这个世界。但是这个该死的Peter没有按照第一个版本世界里的W想的那样用机器去建立桥梁，而是直接毁了另外一个世界。因为两个世界是互相依存的。所以第二个版本的世界发展的也不顺利，到最后还是变的一团糟。22集中peter看到的未来即是第二版本的世界的情况。世界依然分崩离析，女主角也死了，Peter很伤心，W博士也很难过。但是这时候W博士突然领悟到问题出在哪，他决定再将整个世界送回过去。并且想办法将这个版本的世界发生的情景植入到了末日机器里，以便帮助Peter下次正确操作那个机器。可能是把Peter的部分记忆存入了机器。22集中W说过只要让Peter意识到如果他做和上次同样的选择世界将会变成什么样，Peter就会做出正确的操作了。这次，W将末日机器又一次送到过去。后来他担心这次会再出现什么意外，就又研究出了观察者去监督关键事件的发生。观察者应该不是人（也可能是经过特殊处理过的人），他们不能随意穿越时空，但是也不会老死；不过他们应该可以自由穿越于2个世界，或者2个世界的观察者可以自由沟通。他们只是监督关键事件是不是按照第二版本的世界那样发展，以免出现什么意外差错，导致再次错失修复世界的机会。W知道过去的自己会担心peter进入机器后会死掉，所以给观察者的一个任务就是要培养过去的自己，让自己舍得PETER去牺牲。但是为了不干扰历史，W告诉观察者不要直接泄露天机。这也是为什么观察者似乎知道过去将来，但是又不知道如果被干预后会发生什么。他们不是神，只是the second people制造出的产物。

第二个版本的世界至此结束。后来人们还是死光光了，世界也完蛋了。这个版本里的人们应该叫the second people吧。
### 三、第二次回溯的世界（世界在观察者的监督下有惊无险的走上正轨）：
然后第三个版本的世界开始发展了。就是我们看到的世界。我们是the third people！

我们的世界一开始一切正常。但是发展到W乙研制药物的时候出差错了。因为观察者的打扰，W乙没有研制成功。另外一个世界里的perter很可能性命不保，也就不能完成他的使命了（操作机器建立桥梁）。这时候观察者就开始发挥他们的作用了。结果是W博士提前把peter带了过来。但是因为时间提前了，导致他们落入了湖里。按照正常的历史发展他们过来后不应该落进去的。观察者救了他们。但可能由此造出了一个临时性的第三世界。以后的故事就不用我啰嗦了。历史按照稍微有点偏差的样子继续发展。peter进入机器后并没有穿越到未来，而是看到了the second people植入到机器里的未来场景，也就是上一个版本的世界中自己的记忆。这下他知道该怎么操作机器了。至此故事圆满结束。peter也消失了（也可能是观察者救他时产生的临时性的第三世界消失了）。毕竟这个世界如果没有观察者的干扰，peter应该是死了。

我理解的大概就是这样吧。不过还是有很多漏洞无法说通。按理说the first people留下的图里应该没有画着olive的那部分，这部分应该是the second people加上的。因为peter看到的the second people的未来场景时， W对他说olive是作为后备用的。如果是这样的话，第二版本的世界中，peter进入机器前的情节应该和我们看到的当前世界里的情节不一样，而且他进入机器后没有看到未来场景。

有人怀疑W甲有本事研究出观察者这么牛B的东西吗。我觉得既然W乙能在很久前研究出变形人，W甲当然可以在几十年后研究出观察者了。

Peter去哪了？也许在未来的某个时间点在等着他们吧。也可能是去了第三世界里了。
